## 使用方法：
1、找到对应的m3u文件，比如gang_ao_tai.m3u；

2、点进去，右上角有个“raw"按钮，点一下；

3、然后把浏览器里的URL拷贝出来放到IPTV客户端里即可愉快的观影了；
